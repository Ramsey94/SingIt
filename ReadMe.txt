About this project: 

"SingIt!" Is a music sheet audio analyzer for singing training. This tool helps singers to improve their vocal performance. 

It works by analyzing audio recordings of a singer's performance against a pre-defined score or sheet music. The software uses advanced signal 
processing techniques to compare the singer's pitch, and rhythm to the original score. 

Features:

* It provides feedback to the singer in real-time, allowing them to make adjustments to their performance and improve their skills. 

* The software also store summaries of the singer's progress, enabling the singer to track their progress over time and see their improvement.

* Any music score in MIDI format could be accepted for a training session.


About the development:

This software was created using MATLAB and is a project that may be updated with new features over time.
